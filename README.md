# Amazon CloudFormation API

Amazon CloudFormation is an infrastructure as code service that allows you to model, provision, and manage AWS and third-party resources by treating infrastructure as code. CloudFormation enables you to use a template file to create and delete a collection of resources together as a single unit (a stack), automating the provisioning and updating of your infrastructure in an orderly and predictable fashion.

## APIs

- **Amazon CloudFormation API** - The Amazon CloudFormation API provides programmatic access to create, update, and delete stacks, manage change sets, and work with stack resources and templates.
